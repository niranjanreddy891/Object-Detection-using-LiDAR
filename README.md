# Object-Detection-using-LiDAR

This repo refers to object detection using LiDAR data specifically LAS and LAZ formats.

Main libraries used in this project: 

    1. laspy
    
    2. Keras
    
    3. tensorflow
    
    4. numpy
    
    5. scipy
    
    6. pcl
